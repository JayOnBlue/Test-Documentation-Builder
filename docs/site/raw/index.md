## Overview

This documentation describes the sample Order Management feature in `force-app/`. It is not written by
hand — it is generated automatically every time metadata changes are pushed to `main`, then published to
this GitHub Pages site.

## Prerequisites

- None. This page always exists as the landing page for business/use-case documentation.

## Steps to Navigate

1. Use the tabs above to move between documentation categories.
2. Use the sidebar on the left to jump to a specific feature within a category.
3. Press **/** anywhere to search across every page.
4. Use the **Technical Docs** link in the top bar to see the component-level view of the same codebase —
   classes, objects, flows, and how they depend on each other.
