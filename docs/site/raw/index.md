## Overview

This documentation describes the sample Order Management feature in `force-app/`. It is not written by
hand — it is generated automatically every time metadata changes are pushed to `main`, then published to
this GitHub Pages site.

```callout
type: before
Pick the role that matches what you're here to do — it filters the sidebar down to the sections
that matter for that role. You can always switch back to "Everyone" later.
```

## Steps to Navigate

1. Use the header buttons to move between Business Docs, Changelog, and Technical Reference.
2. Use the sidebar on the left to jump to a specific feature within a category.
3. Press **/** anywhere to search across every page.
4. Open **Technical Reference** to see the component-level view of the same codebase — classes, objects,
   flows, and how they depend on each other.
